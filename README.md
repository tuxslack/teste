# Script usado no sysresccd para clonar e restaurar imagem de sistema (Windows 10 Pro).

## Informações: <br>

```sh
$ chmod +x teste.sh
$ ./teste.sh -h
```

## Dependências: <br>

* partclone >= 0.2.77
* dd
* sfdisk
* gunzip
* cat
* dmidecode
* lshw
* parted
* gparted ou cfdisk
* mount
* ntfs-3g
* ntfsfix
* sed
* bc
* smartctl
* inxi
* lsblk
* lspci
* split
* cut
* sort
* fdisk
* umount
* date
* mkdir
